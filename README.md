# 📌 Invoice Factoring System (Soroban Smart Contract)

## 📖 Project Description
The Invoice Factoring System is a decentralized application built on the Stellar Soroban platform. It enables businesses to convert unpaid invoices into immediate cash by allowing third parties to finance them.
<img width="1919" height="951" alt="image" src="https://github.com/user-attachments/assets/64e6504d-2454-4861-bb00-8ba541f701ef" />

This project demonstrates how blockchain can improve transparency, trust, and efficiency in financial systems.

---

## ⚙️ What it does
- Allows sellers to create invoices on-chain
- Enables factoring (financing) of invoices by marking them as funded
- Stores invoice details securely on the blockchain
- Provides a simple way to track invoice status

---

## 🚀 Features
- 📄 Create Invoice  
- 💰 Factor Invoice  
- 🔍 View Invoice Details  
- 🔐 Immutable blockchain storage  
- ⚡ Fast & low-cost transactions using Stellar  

---

## 🧠 How it works
1. A seller creates an invoice with ID and amount  
2. The invoice is stored on-chain  
3. A buyer (factor) selects the invoice  
4. The invoice is marked as "factored"  
5. Anyone can check invoice status  

---

## 🛠️ Tech Stack
- Rust  
- Soroban SDK  
- Stellar Blockchain  

---

## 📦 Smart Contract Functions
- `create_invoice(id, seller, amount)`  
- `factor_invoice(id)`  
- `get_invoice(id)`
